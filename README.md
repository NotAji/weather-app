# Weather App

A modern, responsive Weather App built with Node.js, Express, and vanilla HTML/CSS/JS. Users can search for any city and get real-time weather information with dynamic backgrounds and icons.

# Features

Search any city and display weather without reloading the page

Animated background based on weather (sunny, cloudy, rainy)

Weather icons and temperature indicators

responsive layout for desktop and mobile

smooth card animation when displaying weather details

proper error handling: blank input, invalid city, or server errors

# Tech Stack

Backend: Node.js, Express, Axios

Frontend: HTML, CSS, JavaScript

Styling: Google Fonts, custom CSS animations

API: OpenWeatherMap

# Installation

1. Clone the repository

   git@github.com:NotAji/weather-app.git

   cd to backend

2. npm install

3. cd to frontend

   npm install
